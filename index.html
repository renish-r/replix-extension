Install Replix Gmail Extension

Click the button below to download the Replix extension ZIP:

Download Replix.zip

Installation Instructions

  1. Click the button above to download replix-extension.zip.
  2. Extract the ZIP file to a location on your computer.
  3. Open Chrome and go to chrome://extensions/.
  4. Enable Developer mode using the toggle in the top-right corner.
  5. Click Load unpacked and select the replix-extension folder you just extracted.
  6. Open Gmail -> Replix will now be active and ready to use!

Note: Chrome blocks one-click installation outside the Web Store for security, so this manual step is required.
